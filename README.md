## ✨ Key Features

- 📅 **Smart Calendar**: Intuitively track your cycles, fertile windows, and ovulation days.
- 📝 **Daily Logging**: Record symptoms, moods, and energy levels with a beautiful, minimalist interface.
- 📊 **Cycle Insights**: Gain a deeper understanding of your body's patterns over time.
- 🔔 **Discreet Reminders**: Stay prepared with gentle notifications for your upcoming cycle.
- 🎨 **Serene Design**: A soft, feminine aesthetic designed to provide comfort and clarity.
- 🔒 **Privacy First**: Your data stays on your device, ensuring complete confidentiality.

## 🛠️ Tech Stack

- **Framework**: [Expo](https://expo.dev) (SDK 54) with [Expo Router](https://docs.expo.dev/router/introduction/)
- **Core**: React Native & TypeScript
- **State Management**: [Zustand](https://github.com/pmndrs/zustand)
- **Styling**: Native Styles with a custom design system
- **Animations**: [React Native Reanimated](https://docs.swmansion.com/react-native-reanimated/)
- **Icons**: [Lucide React Native](https://lucide.dev) & Ionicons
- **Date Handling**: [Day.js](https://day.js.org/)

## 🚀 Getting Started

### Prerequisites
- Node.js (Latest LTS)
- npm or yarn
- Expo Go app on your phone (for testing)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/blitzbugg/luna-flow.git
   cd luna-flow
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npx expo start
   ```

4. **Run on your device**
   Scan the QR code with your Expo Go app (Android) or Camera app (iOS).
